~~Lernplatform Setup~~

~MySQL~
Standardport
Datenbanken
- sessiondb
- userdb

Create database sessiondb;

Create database userdb;


Für die userdb muss die Tabelle "users" angelegt werden:
Create table users( username VARCHAR(255) NOT NULL PRIMARY KEY, lastname VARCHAR(255),
	surname VARCHAR(255),	email VARCHAR(255), password VARCHAR(255) , role BigInt NOT NULL);

~~MongoDB~~
Standardport


Projekt aufsetzen:
(1) in das Verzeichnis wechseln
(2) npm install
(3) MongoDB starten
(4) mySQL starten
(5) npm run start

Die Lernplatform ist nun mit http://localhost:3000 erreichbar
